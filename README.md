# Hi, I'm Saverio Annunziata 👋

### Front-end Developer | React · React Native · Tailwind CSS

I'm a front-end developer focused on **React**, **React Native**, and digital interfaces designed to be solid, accessible, and genuinely useful for people.

I currently work at **Garage 94**, a company specialized in developing **educational aids for people with special needs**, learning tools, and accessibility-oriented digital solutions.

In my day-to-day work, I contribute to the development and maintenance of the **LIAR** ecosystem, a communicator for **Augmentative and Alternative Communication (AAC)** used by people with complex communication needs.

---

## 🚀 What I do

I mainly work on front-end and mobile development, building real-world applications used every day by users, families, caregivers, and professionals.

More specifically, I contribute to the development of:

* **Android and iOS tablet apps** built with **React Native**
* **Mobile apps for smartphones** built with **React Native**
* **Web dashboards for parents and caregivers** built with **React**
* New features, stability improvements, and performance optimizations
* Maintenance, debugging, and user experience improvements
